# Skin-Disorder-Prediction
# 🩺 Skin Disorder Prediction using Machine Learning

A Machine Learning project that predicts different types of skin disorders using clinical and histopathological features. The project performs data preprocessing, exploratory data analysis (EDA), feature engineering, model training, model comparison, and prediction to assist in the early identification of skin diseases.

---

## 📌 Project Overview

Skin diseases often exhibit similar symptoms, making diagnosis challenging. This project leverages Machine Learning algorithms to classify skin disorders based on patient clinical and histopathological data.

The project includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Visualization
- Model Training
- Model Evaluation
- Model Comparison
- Prediction of Skin Disease Classes

---

## 🎯 Problem Statement

The objective is to build a predictive model capable of classifying different skin diseases using patient clinical and histopathological attributes.

The project also compares multiple Machine Learning algorithms and recommends the best-performing model for deployment.

## 📊 Dataset Information

The dataset contains **34 attributes**, including:

- Clinical Features
- Histopathological Features
- Age
- Family History

Target Variable:
- Skin Disease Class

Diseases include:

- Psoriasis
- Seborrheic Dermatitis
- Lichen Planus
- Pityriasis Rosea
- Chronic Dermatitis
- Pityriasis Rubra Pilaris
